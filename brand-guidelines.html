  • Spacing: 0.5x icon width between elements
  • Minimum Size: 120px width (horizontal), 64px (icon-only)

Typography

Font Family: Inter

Modern, clean, highly legible sans-serif font optimized for digital interfaces.

Heading 1 - 48px Bold
Heading 2 - 36px Semibold
Heading 3 - 24px Semibold
Body Text - 16px Regular. This is how regular paragraph text should look in the Lunotes interface. It should be easy to read and comfortable for extended reading sessions.
Small Text - 14px Regular. Used for captions, labels, and secondary information.

Font Weights

  • Regular (400): Body text, descriptions
  • Medium (500): Subheadings, labels
  • Semibold (600): Section headers
  • Bold (700): Page titles, primary headings
  • Black (900): Brand name, hero text

Voice & Tone

Brand Voice

Friendly Professional - Approachable yet competent, helpful without being condescending.

Characteristics:

  • Helpful: Provide clear, actionable guidance
  • Confident: Use definitive language when appropriate
  • Warm: Use inclusive, welcoming language
  • Clear: Avoid jargon and technical terms
